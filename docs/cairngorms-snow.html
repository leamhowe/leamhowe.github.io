Cairngorm Snow

A snow persistence dataset for the Cairngorms National Park, Scotland

About This Data

This data was generated using the SPUN deep learning model for mapping snow cover from Sentinel-2 imagery. The model was used to create Snow Cover Fraction (SCF) maps for the entire Sentinel-2 archive, all of these were combined and normalised by cloud-free view to give an estimate of snow persistence.
